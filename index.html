For, you guessed it, the plugin file.

edit as u like... jus be careful while editing...

Config Options :


Mandatory as said, compulsory

NONE

Optional has defaults

sortOrder: '<default order for default column>'
default: 'ASC' 
sortCol: '<default column to sort (0 indexed columns)>'
default: 0 
sortAll: '<whether to sort all columns>'
default: true

Usage :


HTML simple table tag, with data-toggle="sort" attr

<table data-toggle="sort">
</table>

JS simple plugin call, with mandatory options

$("table").sorter();
needed only when data-toggle="sort" is not defined

Examples :


1. simple, basic

Code
<table id="table1" class="table table-bordered ">
  <caption>Table Caption</caption>
  <thead>
    <tr>
      <th>#</th>
      <th>First Name</th>
      <th>Last Name</th>
      <th>Username</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>Mark</td>
      <td>Otto</td>
      <td>@mdo</td>
    </tr>
    <tr>
      <td>2</td>
      <td>Jacob</td>

      <td>@fat</td>
    </tr>
    <tr>
      <td>3</td>
      <td>Larry</td>
      <td>the Bird</td>
      <td>@twitter</td>
    </tr>
  </tbody>
</table>
